# 🔍 Subdomain Sentinel

A modern **Subdomain Enumeration Tool with Live Status Check** built using a fast and scalable frontend architecture.

---

## 🚀 Features

* 🌐 Subdomain enumeration
* ⚡ Live domain status checking
* 🎯 Fast and responsive UI
* 📊 Clean and minimal dashboard
* 🔒 Built for cybersecurity enthusiasts

---

## 🛠️ Tech Stack

* **Frontend:** Vite + TypeScript
* **Styling:** Tailwind CSS
* **Tooling:** Node.js

---

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/AftabHusain8/subdomain-sentinel.git
cd subdomain-sentinel
```

Install dependencies:

```bash
npm install
```

---

## ▶️ Running the Project

```bash
npm run dev
```

Then open:

```
http://localhost:5173
```
